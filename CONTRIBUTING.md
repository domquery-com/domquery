# Contributing Guidelines

Thank you for contributing to the DomQuery project!

## How to Contribute

### Bug Reports
- Please create a bug report in [GitHub Issues](https://github.com/domquery-com/domquery/issues)
- Please use the bug report template
- Provide as much detail as possible

### Feature Suggestions
- Please create a feature suggestion in [GitHub Issues](https://github.com/domquery-com/domquery/issues)
- Please use the feature request template
- Clearly explain the use case

### Pull Requests
1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

## Code Style
- Follow existing code style
- Write meaningful commit messages
- Add tests when possible

## Asking Questions
- Please ask questions in [GitHub Discussions](https://github.com/domquery-com/domquery/discussions)
- The community will answer. Other users are also welcome to help!
- Use Issues only for bugs and feature suggestions

## Code of Conduct
- Be respectful to each other
- Provide constructive feedback
- Help each other

Thank you!
