# 기여 가이드라인

DomQuery 프로젝트에 기여해주셔서 감사합니다!

## 기여 방법

### 버그 리포트
- [GitHub Issues](https://github.com/domquery-com/domquery/issues)에 버그 리포트를 작성해주세요
- 버그 리포트 템플릿을 사용해주세요
- 가능한 한 상세한 정보를 제공해주세요

### 기능 제안
- [GitHub Issues](https://github.com/domquery-com/domquery/issues)에 기능 제안을 작성해주세요
- 기능 요청 템플릿을 사용해주세요
- 사용 사례를 명확히 설명해주세요

### Pull Requests
1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

## 코드 스타일
- 기존 코드 스타일을 따르세요
- 의미 있는 커밋 메시지를 작성해주세요
- 테스트를 추가해주세요 (가능한 경우)

## 질문하기
- [GitHub Discussions](https://github.com/domquery-com/domquery/discussions)에서 질문해주세요
- 커뮤니티가 답변해드립니다. 다른 사용자들도 도와주세요!
- Issues는 버그와 기능 제안에만 사용해주세요

## 행동 강령
- 서로 존중하기
- 건설적인 피드백 제공
- 도움을 주고받기

감사합니다!
