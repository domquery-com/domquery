# Frequently Asked Questions (FAQ)

## 일반 질문 (General Questions)

### Q: DomQuery는 다른 DOM 조작 라이브러리와 호환되나요?
**A:** DomQuery는 유사한 API 패턴을 제공하지만, 완전히 호환되지는 않습니다. 기존 코드를 DomQuery로 마이그레이션할 때 일부 수정이 필요할 수 있습니다.

### Q: DomQuery는 어떤 브라우저를 지원하나요?
**A:** DomQuery는 모든 모던 브라우저를 지원합니다. IE는 제한적으로 지원됩니다. 자세한 내용은 [공식 사이트](https://domquery.com/?introduction)를 참고하세요.

### Q: DomQuery를 사용하려면 npm이 필요한가요?
**A:** 아니요. npm 설치, CDN, 또는 직접 다운로드 모두 가능합니다. 자세한 내용은 [README.md](README.md)를 참고하세요.

## 기술 질문 (Technical Questions)

### Q: WebView 환경에서 히스토리 관리가 자동으로 되나요?
**A:** 네, DomQuery Alert는 기본적으로 히스토리를 자동으로 관리합니다. `History: false`로 설정하면 비활성화할 수 있습니다.

### Q: file:// URL에서 Ajax가 작동하나요?
**A:** 네, DomQuery Ajax는 file:// URL을 자동으로 처리합니다. WebView Bridge도 자동으로 지원합니다.

### Q: 681개 옵션이 너무 많은 것 같은데, 모두 사용해야 하나요?
**A:** 아니요. 대부분의 옵션은 선택사항이며, 기본값으로도 충분히 사용할 수 있습니다. 필요한 옵션만 선택적으로 사용하세요.
