# Code of Conduct

## Our Pledge

Welcome to the DomQuery community!

We strive to create an open and welcoming community.
All participants must follow this Code of Conduct.

## Expected Behavior

- **Be respectful**: Respect all opinions and experiences
- **Constructive feedback**: Criticize constructively, praise freely
- **Help each other**: Answer questions and ask for help
- **Respect diversity**: Welcome people with diverse backgrounds and experiences

## Unacceptable Behavior

- Harassment, discriminatory remarks, sexual harassment
- Aggressive or insulting comments
- Disclosure of personal information
- Other inappropriate behavior

## Enforcement

If you find inappropriate behavior, please contact the project maintainers.
Appropriate action will be taken.

## Contributing

If you have suggestions to improve this Code of Conduct, please propose them in Issues.

Thank you!
